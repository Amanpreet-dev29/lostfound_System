LOST AND FOUND SYSTEM

🚀 Features
	•	📝 Post lost items with details (title, description, category, contact)
	•	📦 View recently added items
	•	🔍 Search items by keyword and category
	•	🗑️ Delete items
	•	👤 User registration and login system
	•	📊 Real-time data interaction with database



🛠️ Tech Stack

Frontend
	•	HTML
	•	CSS
	•	JavaScript (Fetch API)

Backend
	•	Node.js
	•	Express.js

Database
	•	MySQL



⚙️ How It Works
	1.	User interacts with the frontend (forms, search, etc.)
	2.	JavaScript sends requests to the backend using Fetch API
	3.	Express server processes requests and runs SQL queries
	4.	MySQL database stores and retrieves data
	5.	Response is sent back and displayed dynamically on UI
